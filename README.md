# Chocolate Sales Dashboard(Power BI)

## Project Overview
This project is an interactive sales dashboard created using Power BI.
It provides insights into sales performance, profit analysis, shipment distribution,
top products, and top salespersons.

## Tools Used
- Power BI
- Excel / CSV Dataset
- DAX Measures

## Key Features
- Total Sales, Profit, Shipment distribution
- CY vs PY Comparison
- Product-wise and Geo-wise Analysis
- Top Salespersons Performance
- Profit Percentage Indicators

## Conclusion
This Power BI dashboard provides a clear and interactive view of sales performance, profit trends, and product insights. It helps stakeholders quickly identify top-performing products, regions, and salespersons, supporting better data-driven decisions.
